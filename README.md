Title
# Restaurant Recommendation System
Description
This is a Flask-based Restaurant Recommendation System project developed using Python.
It helps users find the best restaurants based on ratings and preferences.
Technologies Used
## Technologies Used
- Python
- Flask
- HTML
- CSS
- Bootstrap
- Pandas
Features
## Features
- Attractive UI
- Restaurant recommendations
- Search functionality
- Responsive design
How to Run
## How to Run

1. Install Python
2. Install Flask
3. Run:
python app.py
